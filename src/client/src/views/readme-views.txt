welcomePage.js       - The landing page for Map-Recallis; displays the logo, welcome message, and navigation links for login, register, guest access, and user list.
loginPage.js         - Renders the login form for sign-in; shows login-status, handles login authentication.
registerPage.js      - Renders the registration to create an account; registration logic 
gettingStartedPage.js- Lets users start a new mind map project by entering a title, prompt, or uploading a file; passes data to the mind map generator.
generateMapPage.js   - Fetches mindmap data from the backend based on user input; displays the generated mindap using the component from generateMindmap.js.
userListPage.js      - Displays a list of all registered users; used it mainly testing registration success and login.
